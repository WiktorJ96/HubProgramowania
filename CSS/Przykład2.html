Zaawansowane techniki CSS

1. CSS Grid Layout

CSS Grid to potężny system układu dwuwymiarowego, który pozwala na tworzenie złożonych layoutów strony.

Przykład użycia CSS Grid:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 20px;
}

.grid-item {
  grid-column: span 2;
}

@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}

2. CSS Custom Properties (zmienne CSS)

Zmienne CSS pozwalają na dynamiczne modyfikowanie stylów i tworzenie bardziej elastycznych arkuszy stylów.

Przykład zaawansowanego użycia zmiennych CSS:

:root {
  --main-color: #007bff;
  --text-color: #333;
}

@media (prefers-color-scheme: dark) {
  :root {
    --main-color: #4a90e2;
    --text-color: #f4f4f4;
  }
}

.button {
  background-color: var(--main-color);
  color: var(--text-color);
  transition: all 0.3s ease;
}

.button:hover {
  --main-color: #0056b3;
}

3. CSS Animations i Keyframes

CSS Animations pozwalają na tworzenie złożonych animacji bez użycia JavaScript.

Przykład zaawansowanej animacji:

@keyframes rotate-scale {
  0% {
    transform: rotate(0deg) scale(1);
  }
  50% {
    transform: rotate(180deg) scale(1.5);
  }
  100% {
    transform: rotate(360deg) scale(1);
  }
}

.animated-element {
  animation: rotate-scale 3s ease-in-out infinite;
}

4. CSS Shapes

CSS Shapes pozwalają na tworzenie niestandardowych kształtów dla opływania tekstu.

Przykład użycia CSS Shapes:

.circle-text {
  width: 300px;
  height: 300px;
  shape-outside: circle(50%);
  float: left;
  margin-right: 20px;
}

.custom-shape {
  shape-outside: polygon(0 0, 100% 0, 100% 75%, 75% 100%, 0 100%);
  clip-path: polygon(0 0, 100% 0, 100% 75%, 75% 100%, 0 100%);
  width: 200px;
  height: 200px;
  float: left;
  background: #007bff;
}

5. CSS Filters

Filtry CSS pozwalają na stosowanie efektów graficznych do elementów.

Przykład zaawansowanego użycia filtrów:

.image-filter {
  filter: grayscale(50%) blur(2px) brightness(110%);
  transition: filter 0.3s ease;
}

.image-filter:hover {
  filter: grayscale(0%) blur(0) brightness(100%);
}
Wskazówka: Pamiętaj o wydajności przy stosowaniu zaawansowanych technik CSS. Niektóre efekty mogą być kosztowne obliczeniowo, szczególnie na urządzeniach mobilnych.

6. CSS Mask

CSS Mask pozwala na tworzenie masek dla elementów, kontrolując ich przezroczystość.

Przykład użycia CSS Mask:

.masked-element {
  width: 300px;
  height: 200px;
  background-image: url('image.jpg');
  -webkit-mask-image: linear-gradient(to right, transparent, black);
  mask-image: linear-gradient(to right, transparent, black);
}

7. CSS Blend Modes

Tryby mieszania CSS pozwalają na tworzenie efektów nakładania kolorów i obrazów.

Przykład CSS Blend Modes:

.blend-element {
  background-image: url('background.jpg');
  background-color: #3498db;
  background-blend-mode: multiply;
}

.text-blend {
  mix-blend-mode: difference;
  color: white;
}

8. CSS Scroll Snap

CSS Scroll Snap pozwala na kontrolowanie zachowania przewijania, tworząc efekt "przyciągania" do określonych punktów.

Przykład CSS Scroll Snap:

.container {
  scroll-snap-type: y mandatory;
  overflow-y: scroll;
  height: 100vh;
}

.section {
  scroll-snap-align: start;
  height: 100vh;
}

9. CSS Logical Properties

Logiczne właściwości CSS pozwalają na tworzenie layoutów niezależnych od kierunku pisma.

Przykład użycia logicznych właściwości:

.box {
  margin-inline: auto;
  padding-block: 1rem;
  border-inline-start: 2px solid black;
}

.rtl-text {
  direction: rtl;
}

10. CSS Houdini

CSS Houdini to zestaw niskopoziomowych API, które dają programistom większą kontrolę nad silnikiem renderującym przeglądarki.

Przykład użycia CSS Paint API (część Houdini):

CSS.paintWorklet.addModule('myPaintWorklet.js');

.custom-background {
  background-image: paint(myCustomPaint);
}
Wskazówka: Zaawansowane techniki CSS często wymagają sprawdzenia kompatybilności z przeglądarkami. Zawsze testuj swoje rozwiązania w różnych środowiskach i stosuj fallbacki dla starszych przeglądarek.
