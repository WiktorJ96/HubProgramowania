Rozszerzony i szczegółowy przewodnik po CSS

1. Podstawy CSS

CSS (Cascading Style Sheets) to język arkuszy stylów używany do opisywania wyglądu strony napisanej w języku znaczników, takim jak HTML. CSS pozwala na oddzielenie struktury dokumentu od jego prezentacji.

Kluczowe koncepcje:

  • Selektory: Określają, które elementy HTML mają być stylizowane.
  • Właściwości: Definiują, co dokładnie ma być zmienione (np. kolor, rozmiar).
  • Wartości: Określają konkretne ustawienia dla właściwości.

Przykład podstawowej składni CSS:

selector {
    property: value;
}

Sposoby dodawania CSS do HTML:

  1. Inline CSS: Dodawanie stylów bezpośrednio do elementów HTML.
  2. Internal CSS: Umieszczanie CSS w sekcji <style> w <head> dokumentu HTML.
  3. External CSS: Linkowanie zewnętrznego pliku CSS.

2. Selektory CSS

Selektory CSS określają, które elementy HTML mają być stylizowane. Istnieje wiele typów selektorów, które pozwalają na precyzyjne celowanie w elementy.

Podstawowe typy selektorów:

  • Selektor elementu: p { color: blue; }
  • Selektor klasy: .klasa { font-size: 16px; }
  • Selektor ID: #id { background-color: yellow; }
  • Selektor atrybutu: [type="text"] { border: 1px solid gray; }

Zaawansowane selektory:

  • Selektory potomków: div p { margin-bottom: 10px; }
  • Selektory dzieci: ul > li { list-style-type: square; }
  • Selektory rodzeństwa: h1 + p { font-weight: bold; }
  • Pseudoklasy: a:hover { text-decoration: underline; }
  • Pseudoelementy: p::first-line { font-variant: small-caps; }

3. Box Model i Layout

Model pudełkowy (Box Model) jest fundamentalną koncepcją w CSS, która opisuje, jak elementy HTML są renderowane jako prostokątne "pudełka" z zawartością, paddingiem, obramowaniem i marginesem.

Komponenty Box Model:

  • Content: Zawartość elementu (tekst, obrazy, itp.)
  • Padding: Przestrzeń między zawartością a obramowaniem
  • Border: Linia otaczająca padding i zawartość
  • Margin: Przestrzeń poza obramowaniem

Przykład użycia Box Model:

div {
    width: 300px;
    padding: 20px;
    border: 1px solid black;
    margin: 10px;
}

Właściwość box-sizing:

box-sizing: border-box; sprawia, że padding i border są wliczane w całkowitą szerokość i wysokość elementu.

Układy (Layouts):

  • Display: Kontroluje zachowanie elementu (block, inline, inline-block, flex, grid)
  • Position: Określa metodę pozycjonowania elementu (static, relative, absolute, fixed)
  • Float: Pozwala elementom "pływać" w lewo lub prawo
  • Flexbox: Jednowymiarowy system układu
  • Grid: Dwuwymiarowy system układu

4. Responsywne projektowanie

Responsywne projektowanie to podejście, które sprawia, że strony internetowe dobrze wyglądają na różnych urządzeniach i rozmiarach ekranów.

Kluczowe techniki:

  • Media Queries: Pozwalają na zastosowanie różnych stylów w zależności od cech urządzenia.
  • Fluid Grids: Używanie procentowych szerokości zamiast stałych.
  • Elastyczne obrazy: Skalowanie obrazów w zależności od rozmiaru kontenera.

Przykład Media Query:

@media screen and (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

Jednostki względne:

  • em: Relatywna do rozmiaru czcionki rodzica
  • rem: Relatywna do rozmiaru czcionki elementu root (zwykle <html>)
  • vw/vh: Relatywna do szerokości/wysokości viewport

5. Flexbox i Grid

Flexbox i Grid to potężne systemy układu w CSS, które znacznie ułatwiają tworzenie złożonych i responsywnych layoutów.

Flexbox:

Jednowymiarowy system układu, idealny do tworzenia elastycznych kontenerów.

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    flex: 1;
}

Grid:

Dwuwymiarowy system układu, doskonały do tworzenia złożonych layoutów strony.

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

.item {
    grid-column: span 2;
}

6. Animacje i Przejścia

CSS oferuje potężne narzędzia do tworzenia animacji i płynnych przejść, co może znacznie poprawić UX strony.

Przejścia (Transitions):

.button {
    background-color: blue;
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: red;
}

Animacje:

@keyframes slide-in {
    from { transform: translateX(-100%); }
    to { transform: translateX(0); }
}

.animated-element {
    animation: slide-in 1s ease-out;
}

7. Zaawansowane techniki CSS

Poznanie zaawansowanych technik CSS pozwala na tworzenie bardziej złożonych i interaktywnych interfejsów.

CSS Variables (Custom Properties):

:root {
    --main-color: #3498db;
}

.button {
    background-color: var(--main-color);
}

CSS Shapes:

.circle {
    width: 100px;
    height: 100px;
    border-radius: 50%;
    shape-outside: circle();
}

CSS Filters:

img {
    filter: grayscale(50%) blur(2px);
}

8. Typografia w CSS

Efektywne wykorzystanie typografii jest kluczowe dla czytelności i estetyki strony. CSS oferuje wiele narzędzi do kontrolowania wyglądu tekstu.

Podstawowe właściwości typograficzne:

  • font-family: Określa rodzinę czcionek
  • font-size: Kontroluje rozmiar czcionki
  • font-weight: Ustawia grubość czcionki
  • line-height: Kontroluje wysokość linii
  • letter-spacing: Ustawia odstępy między literami
  • text-align: Kontroluje wyrównanie tekstu

Zaawansowane techniki typograficzne:

body {
    font-family: 'Roboto', sans-serif;
    font-size: 16px;
    line-height: 1.5;
}

h1 {
    font-size: 2.5em;
    font-weight: 700;
    letter-spacing: -0.5px;
}

p {
    max-width: 60ch; /* Optymalna długość linii dla czytelności */
    text-align: justify;
    hyphens: auto; /* Automatyczne dzielenie wyrazów */
}

@font-face {
    font-family: 'CustomFont';
    src: url('path/to/font.woff2') format('woff2');
    font-display: swap; /* Optymalizacja ładowania czcionek */
}

9. Efekty wizualne i grafika

CSS umożliwia tworzenie zaawansowanych efektów wizualnych bez konieczności używania grafik rastrowych.

Gradienty:

.gradient-bg {
    background: linear-gradient(45deg, #f3ec78, #af4261);
}

.radial-gradient {
    background: radial-gradient(circle, #f3ec78, #af4261);
}

Cienie:

.box-shadow {
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
}

.text-shadow {
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}

Transformacje:

.transformed {
    transform: rotate(45deg) scale(1.2) translateX(20px);
}

Maski i klipy:

.masked-element {
    -webkit-mask-image: url('mask.svg');
    mask-image: url('mask.svg');
}

.clipped-element {
    clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
}

10. Optymalizacja i wydajność CSS

Optymalizacja CSS jest kluczowa dla zapewnienia szybkiego ładowania i płynnego działania strony.

Techniki optymalizacji:

  • Minimalizacja CSS
  • Używanie skróconych właściwości
  • Unikanie nadmiarowych selektorów
  • Optymalizacja kaskady
  • Lazy loading dla CSS niekrytycznego

Przykład optymalizacji:

/* Przed optymalizacją */
.box {
    margin-top: 10px;
    margin-right: 15px;
    margin-bottom: 10px;
    margin-left: 15px;
    border-width: 1px;
    border-style: solid;
    border-color: black;
}

/* Po optymalizacji */
.box {
    margin: 10px 15px;
    border: 1px solid black;
}

Critical CSS:

Technika polegająca na wbudowaniu krytycznego CSS bezpośrednio w HTML, aby przyspieszyć początkowe renderowanie strony.

11. Metodologie organizacji CSS

Stosowanie metodologii CSS pomaga w utrzymaniu czystego, skalowalnego i łatwego w zarządzaniu kodu.

BEM (Block Element Modifier):

/* Block component */
.btn {}

/* Element that depends upon the block */ 
.btn__price {}

/* Modifier that changes the style of the block */
.btn--orange {} 
.btn--big {}

OOCSS (Object Oriented CSS):

Skupia się na oddzieleniu struktury od skórki i kontenera od zawartości.

SMACSS (Scalable and Modular Architecture for CSS):

Kategoryzuje reguły CSS na: Base, Layout, Module, State i Theme.

Atomic CSS:

Polega na tworzeniu małych, jednozadaniowych klas.

.mt-10 { margin-top: 10px; }
.text-center { text-align: center; }
.bg-primary { background-color: #007bff; }

12. CSS i dostępność

Tworzenie dostępnych stron internetowych jest kluczowe dla zapewnienia równego dostępu do informacji dla wszystkich użytkowników.

Kluczowe aspekty dostępności w CSS:

  • Odpowiedni kontrast kolorów
  • Czytelne rozmiary czcionek
  • Responsywny design
  • Właściwe użycie focus states
  • Ukrywanie treści w sposób dostępny

Przykład dostępnego fokusa:

a:focus {
    outline: 2px solid #4A90E2;
    outline-offset: 2px;
}

.visually-hidden {
    position: absolute;
    width: 1px;
    height: 1px;
    padding: 0;
    margin: -1px;
    overflow: hidden;
    clip: rect(0, 0, 0, 0);
    white-space: nowrap;
    border: 0;
}
Wskazówka: Pamiętaj, że CSS jest potężnym narzędziem, ale zawsze należy myśleć o użytkownikach końcowych. Staraj się tworzyć nie tylko estetyczne, ale również funkcjonalne, wydajne i dostępne strony internetowe.
Wskazówka: Regularnie ćwicz i eksperymentuj z różnymi właściwościami CSS. Tworzenie małych projektów lub wyzwań CodePen może znacznie przyspieszyć naukę i zrozumienie CSS.
